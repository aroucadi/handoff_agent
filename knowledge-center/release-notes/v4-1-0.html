ClawdView v4.1.0 Release Notes

January 2026 — Portfolio scoring revamp, Hub connector marketplace, WCAG 2.1 AA compliance, and GraphQL API beta.

🆕 New Features

Portfolio Scoring Engine Overhaul (Portfolios)

The scoring engine has been completely rewritten to support:

  • Weighted criteria with configurable importance levels (previously all criteria had equal weight)
  • Scoring templates that can be shared across portfolios and organizations
  • Scenario comparison: Score the same portfolio under different criteria weights to model strategic shifts
  • Scoring history: Track how initiative scores change over time as data updates
  • Bulk scoring: Score up to 100 initiatives simultaneously with committee collaboration mode

Hub Connector Marketplace

A self-service marketplace for discovering, installing, and configuring Hub connectors. Key features:

  • Browse 100+ connectors with ratings, reviews, and use case examples
  • One-click installation with guided setup wizard
  • Partner-built connectors alongside ClawdView-certified ones
  • Connector health monitoring with uptime SLA indicators

WCAG 2.1 AA Compliance

Full accessibility compliance achieved across all ClawdView modules:

  • Keyboard navigation for all interactive elements including board cards and Gantt tasks
  • Screen reader compatibility (tested with NVDA, JAWS, and VoiceOver)
  • Color contrast ratios meeting AA standards (minimum 4.5:1 for text)
  • Focus indicators visible on all interactive components
  • ARIA labels and roles on custom widgets

GraphQL API Beta

A new GraphQL endpoint for complex, multi-resource queries. See API Integration Guide for details. Key capabilities in beta:

  • Query projects with nested tasks, resources, and financials
  • Portfolio items with scoring criteria and history
  • Board cards with custom fields and connection data
  • WebSocket-based subscriptions for real-time card movement events

🔧 Improvements

  • AgilePlace: Dependency lines now show risk level coloring (green = on track, amber = at risk, red = blocked)
  • PPM Pro: Baseline comparison view shows delta overlay on the Gantt chart
  • Portfolios: Investment scenario modeling supports up to 10 scenarios (previously 3)
  • Hub: Azure DevOps connector now supports Board-level sync (previously limited to Backlog items)
  • Performance: Dashboard rendering speed improved by 30% through data caching layer

🐛 Bug Fixes

  • Fixed: AgilePlace automation rules firing twice when card was moved by another automation (cascade loop)
  • Fixed: PPM Pro resource leveling ignored non-working days in custom calendars
  • Fixed: Hub ServiceNow connector timed out on initial sync with >50,000 incidents
  • Fixed: Portfolios roadmap PDF export cut off initiative names longer than 60 characters
  • Fixed: Time entry approval workflow skipped the second-level approver in hierarchy

⚠ Deprecations

  • API v1: All v1 REST endpoints now return deprecation warnings. Full removal planned for v4.3.0 (Q3 2026). Migrate to v2.
  • Classic Board UI: The legacy board renderer (pre-v3.0) has been removed. All boards now use the modern rendering engine.
