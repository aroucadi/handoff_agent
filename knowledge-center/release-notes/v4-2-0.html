ClawdView v4.2.0 Release Notes

March 2026 — AI-powered insights, enhanced board performance, new governance templates, and 12 new Hub connectors.

🆕 New Features

AI-Powered Portfolio Insights (Portfolios)

Powered by Google Gemini, ClawdView now generates natural language insights from your portfolio data. Navigate to any dashboard and click the "AI Insights" button to receive:

  • Portfolio health summary with top 3 risks explained in plain language
  • Resource bottleneck identification with recommended reallocation suggestions
  • Budget forecast analysis with trend-based projections
  • Strategic alignment gaps — initiatives not mapping to any strategic theme
🔑 Requirements

AI Insights requires Enterprise tier and is available in US and EU regions. Data is processed in Google Cloud's us-central1 or europe-west1 regions based on your tenant configuration. No customer data is used for model training.

Board Performance Mode (AgilePlace)

AgilePlace boards with 500+ cards now automatically engage Performance Mode, which:

  • Virtualizes card rendering — only visible cards are in the DOM
  • Reduces initial load time by up to 70% for large boards
  • Enables "infinity scroll" for boards with 1,000+ cards
  • Maintains full search and filter functionality despite virtualization

Governance Templates Library (Portfolios)

12 pre-built governance templates for common scenarios:

  • IT Portfolio Rationalization, M&A Integration, Annual Planning Cycle
  • SAFe Lean Portfolio Management, PRINCE2 Stage-Gate, ITIL Service Portfolio
  • Each template includes stage-gate definitions, scoring criteria, and dashboard configurations

New Hub Connectors

Connector Direction Key Capabilities

Asana Bidirectional Project sync, task mapping, custom field support
Smartsheet Import Sheet import with column mapping
Wrike Bidirectional Project + task sync with folder hierarchy
Workday Import Employee data, cost centers, skills
Google BigQuery Export Analytics data push for custom BI

🔧 Improvements

  • PPM Pro Gantt: 40% faster rendering for projects with 500+ tasks. New mini-map navigation for large plans.
  • Smart Suggest: Improved skill matching algorithm. Now considers recent project experience alongside certified skills.
  • Portfolios Scoring: Support for weighted scoring with must-pass (knockout) criteria. Scoring history preserved for audit.
  • Hub Sync: Retry logic improved with circuit breaker pattern. Failed syncs now provide root cause analysis in the error dashboard.
  • Mobile App: AgilePlace mobile now supports card creation, board filtering, and push notifications for card movement.

🐛 Bug Fixes

  • Fixed: Portfolio roadmap timeline incorrectly displayed cross-year initiatives spanning Q4→Q1
  • Fixed: Hub Jira connector lost custom field mappings after Jira Cloud version upgrades
  • Fixed: PPM Pro resource utilization calculation double-counted shared pool assignments
  • Fixed: AgilePlace board export to CSV truncated card descriptions exceeding 5,000 characters
  • Fixed: SSO SCIM provisioning failed for users with apostrophes in last names

⚠ Breaking Changes

  • API v1 Deprecation: REST API v1 endpoints are now fully deprecated. Migrate to v2 before May 2026. See API Guide.
  • Board Card Limit: Standard tier boards now enforce a hard limit of 500 cards (previously soft warning at 1,000). Use hierarchical board decomposition if hitting this limit.
